redis 学习笔记

一、redis简介

redis是开源的、先进的key-value存储系统。
它的特性如下：

  • redis完全将数据保存在内存里，只用硬盘进行持久化
  • 和其他Key-value存储系统相比，它支持的数据类型更丰富
  • redis可以同步数据到任何数量的从机上

它的优势如下：

  • 非常快。大约每秒110000次写和81000读
  • 支持丰富的数据类型
  • 原子操作
  • 很多实用工具，如高速缓存、消息队列(原生支持发布/订阅机制)、临时数据如sessions,page hit counts等的存储

四、支持的数据类型

  • String

六、HyperLogLog类型

这种特殊的类型是解决统计问题的，事实上它是一种算法的应用。
假如你要统计每日访问网站的IP地址的量，如果放进Sets里然后统计Sets大小，会非常浪费内存(如果访问量很大)。
如果你只是关心最终数据的基数估算值，不要求数据真的存储起来，那用这种类型是比较好的。
这种类型每个Key只用12Kb字节就能得到基数的估算值，估算标准差在0.81%。你最高可以统计2^64个项目
